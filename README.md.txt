## Proje Açıklaması
Bu projede Python programlama dili kullanılarak kullanıcıdan alınan bilgilerle otomatik olarak bir HTML web sayfası oluşturulmuştur. Program çalıştırıldığında kullanıcıdan isim, aldığı dersler ve kısa bir biyografi bilgisi alınmaktadır. Alınan bu bilgiler Python içerisinde string değişkenler kullanılarak HTML formatına dönüştürülmektedir.

Oluşturulan HTML sayfası temel HTML etiketleri kullanılarak hazırlanmıştır. Sayfanın daha düzenli ve okunabilir olması için basit CSS tanımlamaları eklenmiştir. Kullanıcının girdiği ders bilgileri liste yapısı kullanılarak web sayfasında gösterilmektedir.

Proje, Python’da dosya işlemlerinin nasıl yapıldığını göstermek amacıyla hazırlanmıştır. open(), write() ve close() fonksiyonları kullanılarak oluşturulan HTML içeriği index.html adlı dosyaya yazılmaktadır. Program tamamlandığında bu dosya otomatik olarak oluşturulmakta ve tarayıcıda açıldığında kullanıcının girdiği bilgilere göre düzenlenmiş bir web sayfası görüntülenmektedir.


## Kodun Çalışma Mantığı
Program çalıştırıldığında kullanıcıdan isim, aldığı dersler ve biyografi bilgileri input() fonksiyonu ile alınır. Dersler bilgisi split() fonksiyonu kullanılarak liste haline getirilir. HTML sayfası Python içerisinde string değişkenler yardımıyla oluşturulur ve basit CSS eklenir. Oluşturulan HTML içeriği index.html dosyasına yazılır.
